Overview

Rise of P Dindy is a 2D adventure game developed as a university software development project. The game challenges players to explore different levels, overcome obstacles, defeat enemies, and complete objectives through interactive gameplay.

The project was developed collaboratively, providing hands-on experience in game development, software design, and teamwork.

Features
Character movement
Enemy interaction
Multiple game levels
Collision detection
Health system
Interactive gameplay
Game menu
Score/progress system


Technologies
Java 
Git
Unity
